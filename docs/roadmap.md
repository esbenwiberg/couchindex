# Roadmap

## Milestone 1: runnable TV shell

- Android TV project
- Compose for TV navigation
- Home, Browse and Settings destinations
- Remote focus behavior

## Milestone 2: catalogue vertical slice

- TMDb authentication
- Danish availability
- Subscription configuration
- Browse rows and title details
- Provider launch handoff

## Milestone 3: personal state

- Local watchlist
- Recent launches
- Continue Watching as the first row
- Manual watched and remove actions

## Milestone 4: rating enrichment

- IMDb rating import or lookup
- Vote counts
- Additional rating adapter
- Quality-oriented browse filters

## Later investigations

- Media-session playback observation
- Usage-based completion heuristics
- Phone companion
- Trakt or Letterboxd integration
- Recommendations based on personal feedback
